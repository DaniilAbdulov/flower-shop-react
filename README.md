Интернет магазин. Полное CRUD приложение. Авторизация/регистрация. Реализован функционал для просмотра товаров, добавления в избранное/корзину. Реализовано оформление заказа, оплата, отмена, повтор. В панели админа есть возможность добавления нового товара, его изменение и удаление. Можно добавить/удалить категорию, смотреть статистику. Присваивать заказам статус "выдан". В личном кабинете пользователя можно получить список всех заказов и небольшую статистику.
Стек: React, Redux Toolkit, Axios, NodeJS, ExpressJS, Bcrypt, JWT, PostgresSQL.
